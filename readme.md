DMthis
======

Overview
--------
_Never miss an important tweet again._
This is a Twitter bot and site frontend to manage delivery of DMs from statuses by specified users.
For more information, see [README] (app/README).

Directories
-----------
* `app` - DMthis daemon
* `site` - Website
